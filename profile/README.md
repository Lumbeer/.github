# Lumbeer

**Construction & civil-engineering AI tools.**

We build practical AI tooling that turns engineering documents and drawings into
structured, reviewable data — so teams spend less time on manual takeoffs and
transcription, and more time on engineering.

## Focus areas

- 📐 **Drawing & document extraction** — structural drawings, schedules, and plans → structured JSON
- 🏗️ **Quantity takeoff aids** — columns, walls, dimensions, and notes pulled straight from PDFs
- ✅ **LLM-assisted QA** — automated review and local verification of extracted data

## Projects

- **construct-pdf-reader** — extracts grid, columns, walls, dimensions, and notes
  from structural engineering PDF drawings into structured JSON using Gemini,
  with an LLM QA review pass and local verification checks.

---

<sub>Always verify generated output against the source drawings before use in design or analysis.</sub>
